  - else -> return `{"status": "unknown_source", "parse_source": actual_parse_source}`

- Update paper_sources: set `ps.parse_status = "success"` for the active source
- Upsert citations: `_upsert_citations(session, paper, paper_json.get("citations", []))`
- `session.commit()`

- Exception handler: `session.rollback()` then `raise self.retry(exc=exc)`

**Helper functions (module-level, outside the task):**

**`_normalize_s2orc(raw: dict, paper) -> dict:`**
- Handle S2ORC pdf_parse wrapper: `pdf_parse = raw.get("pdf_parse") or raw`
- `body_text = pdf_parse.get("body_text", [])`
- `bib_entries = pdf_parse.get("bib_entries", {})`
- Group consecutive paragraphs with same `(section, sec_num)` tuple into sections using `_make_section()`
- Convert bib_entries to citations via `_bib_entries_to_citations(bib_entries)`
- Merge grobid_citations if present: `raw.get("grobid_citations", [])` -> `_grobid_raw_to_citations()` -> merge
- `_merge_citations(s2orc_cits, grobid_cits)`: combine, grobid citations fill gaps (DOI, raw_text) in S2ORC entries by title match; append any GROBID-only citations
- Return dict with keys: title, abstract, authors, sections, citations, ref_entries, back_matter, parse_source, parse_quality

**`_normalize_mineru(raw: dict, paper) -> dict:`**
- `content_list = raw.get("content_list", [])`
- Iterate content_list: `type=="title"` starts new section (flush previous), `type=="text"` appends to current
- Per Pitfall 2: if zero "title" blocks found, create single section with heading="" and sec_num=None containing all text
- Citations from `raw.get("grobid_citations", [])` via `_grobid_raw_to_citations()`
- Return dict with same keys as _normalize_s2orc

**`_normalize_grobid_fulltext(raw: dict, paper) -> dict:`**
- Sections from `raw.get("grobid_sections", [])` — already in correct shape from parse_pdf_grobid primary mode (Plan 04-01)
- Citations from `raw.get("grobid_citations", [])` via `_grobid_raw_to_citations()`
- Return dict with same keys

**`_make_section(heading: str, sec_num, texts: list[str]) -> dict:`**
- `full_text = " ".join(texts)`
- `paragraphs = [{"text": t, "cite_spans": [], "ref_spans": []} for t in texts if t]`
- Return `{"heading": heading or "", "sec_num": sec_num, "text": full_text, "paragraphs": paragraphs, "token_count": 0}`

**`_flatten_authors(authors_raw: list) -> list[str]:`**
- If `authors_raw` is empty or not a list, return `[]`
- If first item is dict with "first"/"last" keys: `f"{a.get('first','')} {a.get('last','')}".strip()`
- If first item is string: return as-is
- Return list of name strings

**`_authors_from_paper(paper) -> list[str]:`**
- If paper has content with authors key, use those
- Else return `[]`

**`_bib_entries_to_citations(bib_entries: dict) -> list[dict]:`**
- For each `(ref_id, bib)` in `bib_entries.items()`:
- Flatten authors: if `bib["authors"]` contains dicts, convert to `"First Last"` strings (Pitfall 6)
- Return `{"ref_id": ref_id, "title": bib.get("title"), "authors": authors, "year": bib.get("year"), "venue": bib.get("venue"), "doi": bib.get("doi"), "arxiv_id": bib.get("arxiv_id"), "raw_text": bib.get("raw_text")}`

**`_grobid_raw_to_citations(grobid_cits: list[dict]) -> list[dict]:`**
- For each `(i, cit)` in `enumerate(grobid_cits)`:
- Return `{"ref_id": f"BIBREF{i}", "title": cit.get("title"), "authors": cit.get("authors", []), "year": cit.get("year"), "venue": None, "doi": cit.get("doi"), "arxiv_id": None, "raw_text": cit.get("raw_text")}`

**`_merge_citations(primary: list[dict], secondary: list[dict]) -> list[dict]:`**
- Start with primary list
- For each secondary citation: if title matches an existing citation (case-insensitive), merge missing fields (doi, raw_text); else append with new ref_id
- Return merged list

**`_add_token_count(paper_json: dict) -> None:`**
- `import tiktoken` (lazy)
- `enc = tiktoken.get_encoding("cl100k_base")`
- `all_text = " ".join(s.get("text", "") for s in paper_json.get("sections", []))`
- `paper_json["token_count"] = len(enc.encode(all_text)) if all_text.strip() else 0`
- Also set per-section: `sec["token_count"] = len(enc.encode(sec.get("text", ""))) if sec.get("text") else 0`

**`_add_tldr(paper_json: dict) -> None:`**
- `abstract = paper_json.get("abstract") or ""`
- If not abstract.strip(): `paper_json["tldr"] = None; return`
- `sentences = [s.strip() for s in abstract.split(". ") if s.strip()]`
- `tldr_sentences = sentences[:3]`
- `paper_json["tldr"] = ". ".join(tldr_sentences) + "." if tldr_sentences else None`
- Edge case: if abstract doesn't contain ". " at all, use the full abstract as tldr

**`_add_dedup_fingerprint(paper_json: dict) -> None:`**
- title = paper_json.get("title") or ""
- authors = paper_json.get("authors") or []
- year = paper_json.get("year")
- If not title or not authors or not year: `paper_json["dedup_fingerprint"] = None; return`
- `norm_title = re.sub(r"[^a-z0-9]", "", title.lower())`
- first_author = authors[0] if authors else ""
- `last_name = re.sub(r"[^a-z0-9]", "", first_author.split()[-1].lower()) if first_author else ""`
- `raw = f"{norm_title}|{last_name}|{year}"`
- `paper_json["dedup_fingerprint"] = hashlib.sha256(raw.encode()).hexdigest()`

**`_build_src_url(paper) -> str:`**
- If paper.arxiv_id: return `f"https://arxiv.org/abs/{paper.arxiv_id}"`
- If paper.pmc_id: return `f"https://www.ncbi.nlm.nih.gov/pmc/articles/{paper.pmc_id}/"`
- Return ""

**`_check_dedup_and_link(session, paper, paper_json: dict) -> bool:`**
- fp = paper_json.get("dedup_fingerprint")
- If not fp: return False
- Query: `session.execute(text("SELECT canonical_id, arxiv_id, pmc_id FROM papers WHERE content->>'dedup_fingerprint' = :fp AND canonical_id != :cid LIMIT 1"), {"fp": fp, "cid": str(paper.canonical_id)})`
- If match found: create IdMap row linking this paper's IDs to existing canonical_id; `session.merge(id_map_row)`; return True
- Return False

**`_upsert_paper(session, paper, paper_json: dict) -> None:`**
- `from sqlalchemy.dialects.postgresql import insert as pg_insert`
- `from sqlalchemy import func`
- `from app.models import Paper`
- Build values dict with all Paper columns mapped from paper_json and paper object
- `stmt = pg_insert(Paper.__table__).values(...).on_conflict_do_update(index_elements=["canonical_id"], set_={content, token_count, tldr, parse_source, parse_quality, title, abstract, updated_at})`
- `session.execute(stmt)`

**`_upsert_citations(session, paper, citations: list[dict]) -> None:`**
- `from sqlalchemy.dialects.postgresql import insert as pg_insert`
- `from app.models import PaperCitation, IdMap`
- For each citation:
  - Resolve target_paper_id via IdMap lookup on target_arxiv_id then target_doi
  - Skip citations with no target_arxiv_id AND no target_doi (can't upsert without conflict key)
  - `pg_insert(PaperCitation.__table__).values(...).on_conflict_do_update(constraint="uq_paper_citations_source_target_arxiv", set_={"context_text": ..., "target_paper_id": ...})`
  - For citations with target_doi but no target_arxiv_id: use `ON CONFLICT DO NOTHING` (no unique constraint on doi)
  - `session.execute(stmt)`
