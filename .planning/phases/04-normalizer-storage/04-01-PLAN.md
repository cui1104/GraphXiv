**1. Add `extract_fulltext()` to `app/parsers/grobid.py`:**

Add a new function `extract_fulltext(pdf_path: str, timeout: int = 60) -> tuple[list[dict], list[dict]]` that:
- POSTs PDF to `{GROBID_URL}/api/processFulltextDocument` with `files={"input": ("paper.pdf", pdf_bytes, "application/pdf")}` and `data={"includeRawCitations": "1", "consolidateCitations": "0"}`
- On success (status 200), calls `_parse_tei_fulltext_sections(resp.content)` for sections and `_parse_tei_references(resp.content)` for citations (reuse existing function)
- Returns `(sections, citations)` tuple
- Returns `([], [])` on any failure (non-blocking, same as extract_references pattern per D-07)
- Uses `timeout=60` (not 30) because fulltext processing is heavier than reference-only (Pitfall 8)

Add `_parse_tei_fulltext_sections(tei_xml: bytes) -> list[dict]` that:
- Parses TEI XML with `lxml.etree.fromstring(tei_xml)`
- Finds body: `root.find(f".//{{{TEI_NS}}}body")`
- Iterates `body.findall(f"{{{TEI_NS}}}div")` for each section div
- For each div: `sec_num = div.get("n")`, `heading = head_el.text.strip()` from `div.find(f"{{{TEI_NS}}}head")`
- Collects paragraphs from `div.findall(f"{{{TEI_NS}}}p")` using `"".join(p_el.itertext()).strip()`
- Each paragraph dict: `{"text": text, "cite_spans": [], "ref_spans": []}`
- Each section dict: `{"heading": heading, "sec_num": sec_num, "text": full_text, "paragraphs": paras, "token_count": 0}`
- Returns empty list if body is None
