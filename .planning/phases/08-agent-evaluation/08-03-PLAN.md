<shared_conventions>
- **Schema-versioned scores.jsonl**: every row has `scores_schema_version: 1` so future migrations are explicit per D-26.
- **Judge prompt construction**: reads `eval/rubric.md` at runtime and embeds its text verbatim as the judge's system prompt. Do NOT duplicate rubric wording in Python source (Pitfall 7 — prompt drift).
- **Gold data NOT sent to judge** per Anti-Pattern 7: judge sees only question_text + gold_cited_arxiv_ids (for citation_coverage context) + agent's answer_text + agent's tool_call names/arxiv_ids. It does NOT see gold_answer_keywords or any reference answer.
- **Position-bias mitigation** per 08-CONTEXT.md Area of Discretion: for paired judging, the judge sees BOTH condition answers side-by-side for the same question, in an order randomized by `hash(question_id) % 2`. A mapping is recorded in each score row (`presentation_order: ["baseline", "citation_aware"]` or its reverse) so downstream analysis re-associates correctly.
- **One judge call per (question_id) pair**: not per row. So 30 questions → 30 judge calls, each yielding TWO score rows (one per condition). This halves the judge cost vs per-row judging.
- **deterministic_citation_coverage signature**:
  ```python

      # Returns fraction in [0, 1]; fraction of gold cites that appear in any tool_call's arguments (arxiv_id/paper_id/id).
  ```
- **Wilcoxon alternatives**: per dimension, run a two-sided test by default (no pre-registration hypothesis); ALSO report a one-sided `alternative="greater"` for the directional hypothesis "citation_aware > baseline" so FINDINGS.md can be honest about both.
- **Notebook naming**: figures saved to `eval/notebook/figs/fig1_*.png`, `fig2_*.png`, etc., so FINDINGS.md can embed them with relative paths.

    """LLM-as-judge scorer for Phase 8 evaluation (EVAL-03).

    For each question Qxxx that has both baseline + citation_aware success rows in
    eval/results/runs.jsonl, call gpt-4o-mini (D-11) once with BOTH answers (randomized
    order per 08-CONTEXT.md position-bias mitigation) and persist a 4-dimension verdict
    per condition to eval/results/scores.jsonl. Cross-check citation_coverage with a
    deterministic grounding check per D-19 and 08-RESEARCH.md Pattern 5.

    Structured output is enforced via response_format=json_schema with strict=True per
    08-RESEARCH.md Pattern 1.
    """

    import argparse
    import hashlib
    import json
    import logging
    import os
    import random
    import sys
    from collections import defaultdict
    from datetime import datetime, timezone
    from pathlib import Path
    from typing import Callable, Optional

    sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))

    SCORES_SCHEMA_VERSION = 1
    DEFAULT_RUNS_PATH = Path(__file__).parent / "results" / "runs.jsonl"
    DEFAULT_SCORES_PATH = Path(__file__).parent / "results" / "scores.jsonl"
    DEFAULT_QUESTIONS_PATH = Path(__file__).parent / "questions.json"
    DEFAULT_RUBRIC_PATH = Path(__file__).parent / "rubric.md"
    JUDGE_MODEL = "gpt-4o-mini"  # D-11
    JUDGE_TEMPERATURE = 0.0       # D-16
    JUDGE_SEED = 42               # D-16

    logger = logging.getLogger("score")
    logging.basicConfig(level=logging.INFO, format="%(asctime)s %(levelname)s %(message)s")

    JUDGE_SCHEMA = {
        "type": "object",
        "properties": {
            "answer_a": {
                "type": "object",
                "properties": {
                    "answer_correctness": {"type": "integer", "minimum": 1, "maximum": 5},
                    "faithfulness": {"type": "integer", "minimum": 1, "maximum": 5},
                    "citation_coverage": {"type": "integer", "minimum": 1, "maximum": 5},
                    "completeness": {"type": "integer", "minimum": 1, "maximum": 5},
                    "notes": {"type": "string"},
                },
                "required": ["answer_correctness", "faithfulness", "citation_coverage", "completeness", "notes"],
                "additionalProperties": False,
            },
            "answer_b": {
                "type": "object",
                "properties": {
                    "answer_correctness": {"type": "integer", "minimum": 1, "maximum": 5},
                    "faithfulness": {"type": "integer", "minimum": 1, "maximum": 5},
                    "citation_coverage": {"type": "integer", "minimum": 1, "maximum": 5},
                    "completeness": {"type": "integer", "minimum": 1, "maximum": 5},
                    "notes": {"type": "string"},
                },
                "required": ["answer_correctness", "faithfulness", "citation_coverage", "completeness", "notes"],
                "additionalProperties": False,
            },
        },
        "required": ["answer_a", "answer_b"],
        "additionalProperties": False,
    }


    # ---------- Deterministic citation_coverage (D-19, Pattern 5) ----------

    def deterministic_citation_coverage(gold_arxiv_ids: list[str], tool_calls: list[dict]) -> float:
        """Fraction of gold cites that appear in any tool_call's arguments.

        Lookup surfaces checked on each tool_call: 'arxiv_id_hit' (normalized by run_eval),
        and the raw 'arguments' dict keys 'arxiv_id' / 'paper_id' / 'id'.
        Returns a float in [0, 1]; 0.0 if gold_arxiv_ids is empty.
        """
        if not gold_arxiv_ids:
            return 0.0
        hits = set()
        for tc in tool_calls or []:
            for cand in (tc.get("arxiv_id_hit"),):
                if cand:
                    hits.add(str(cand))
            args = tc.get("arguments") or {}
            if isinstance(args, dict):
                for k in ("arxiv_id", "paper_id", "id"):
                    if args.get(k):
                        hits.add(str(args[k]))
        matched = sum(1 for g in gold_arxiv_ids if str(g) in hits)
        return matched / len(gold_arxiv_ids)


    # ---------- Judge prompt ----------

    def _read_rubric(rubric_path: Path = DEFAULT_RUBRIC_PATH) -> str:
        return Path(rubric_path).read_text(encoding="utf-8")


    def _build_judge_prompt(
        question: dict,
        run_a: dict,
        run_b: dict,
        rubric_text: str,
    ) -> list[dict]:
        """Build (system, user) messages for the judge. Does NOT include gold_answer_keywords per Anti-Pattern 7."""
        system_msg = (
            "You are a careful grader of research-paper Q&A answers. Score each answer on the "
            "four 1-5 dimensions below. Output STRICT JSON matching the provided schema. "
            "Do not add commentary outside the schema.\n\n"
            "Rubric:\n" + rubric_text
        )
        # Tool-call summary shown to the judge (names + arxiv_ids hit). Full arguments
        # are omitted to keep prompt short.
        def _tc_summary(tcs: list[dict]) -> list[dict]:
            return [{"name": tc.get("name"), "arxiv_id_hit": tc.get("arxiv_id_hit")} for tc in (tcs or [])]

        user_msg = json.dumps({
            "question_text": question["question_text"],
            "question_type": question["question_type"],
            "gold_cited_arxiv_ids": question.get("gold_cited_arxiv_ids", []),
            "answer_a": {
                "answer_text": run_a.get("answer_text") or "",
                "tool_calls_summary": _tc_summary(run_a.get("tool_calls")),
            },
            "answer_b": {
                "answer_text": run_b.get("answer_text") or "",
                "tool_calls_summary": _tc_summary(run_b.get("tool_calls")),
            },
        }, ensure_ascii=False, indent=2)
        return [
            {"role": "system", "content": system_msg},
            {"role": "user", "content": user_msg},
        ]


    def parse_judge_verdict(raw: str) -> dict:
        """Parse the judge's JSON string and validate it matches JUDGE_SCHEMA at runtime."""
        obj = json.loads(raw)
        if not isinstance(obj, dict) or "answer_a" not in obj or "answer_b" not in obj:
            raise ValueError(f"judge verdict missing answer_a/answer_b: keys={list(obj.keys())}")
        for key in ("answer_a", "answer_b"):
            sub = obj[key]
            for dim in ("answer_correctness", "faithfulness", "citation_coverage", "completeness"):
                v = sub.get(dim)
                if not isinstance(v, int) or not (1 <= v <= 5):
                    raise ValueError(f"judge verdict {key}.{dim}={v!r} not int in [1,5]")
        return obj


    # ---------- Score I/O ----------

    def _load_runs_by_qid_cond(runs_path: Path) -> dict[tuple[str, str], dict]:
        """Return {(question_id, condition): row} for ALL successful rows (error == None)."""
        out: dict[tuple[str, str], dict] = {}
        with open(runs_path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                row = json.loads(line)
                if row.get("error"):
                    continue
                out[(row["question_id"], row["condition"])] = row
        return out


    def _load_done_questions(scores_path: Path) -> set[str]:
        """Questions that already have BOTH conditions scored (error-free)."""
        if not Path(scores_path).exists():
            return set()
        by_q: dict[str, set[str]] = defaultdict(set)
        with open(scores_path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                try:
                    row = json.loads(line)
                except json.JSONDecodeError:
                    continue
                if row.get("error"):
                    continue
                by_q[row["question_id"]].add(row["condition"])
        return {q for q, conds in by_q.items() if {"baseline", "citation_aware"}.issubset(conds)}


    def _append_row(scores_path: Path, row: dict) -> None:
        Path(scores_path).parent.mkdir(parents=True, exist_ok=True)
        with open(scores_path, "a", encoding="utf-8") as f:
            f.write(json.dumps(row, ensure_ascii=False) + "\n")
            f.flush()


    # ---------- Per-question scoring ----------

    def _presentation_order(question_id: str) -> list[str]:
        """Deterministic-per-question order. hash(qid) % 2 == 0 -> baseline first; else citation_aware first."""
        h = int(hashlib.sha256(question_id.encode("utf-8")).hexdigest(), 16)
        return ["baseline", "citation_aware"] if h % 2 == 0 else ["citation_aware", "baseline"]


    def score_question(
        question: dict,
        run_baseline: dict,
        run_citation_aware: dict,
        client,
        rubric_text: str,
    ) -> list[dict]:
        """Return two score rows (one per condition) for a single question_id."""
        order = _presentation_order(question["question_id"])  # e.g. ["citation_aware", "baseline"]
        runs_by_cond = {"baseline": run_baseline, "citation_aware": run_citation_aware}
        run_a = runs_by_cond[order[0]]
        run_b = runs_by_cond[order[1]]

        messages = _build_judge_prompt(question, run_a, run_b, rubric_text)
        resp = client.chat.completions.create(
            model=JUDGE_MODEL,
            temperature=JUDGE_TEMPERATURE,
            seed=JUDGE_SEED,
            max_tokens=1024,
            response_format={
                "type": "json_schema",
                "json_schema": {"name": "judge_verdict", "schema": JUDGE_SCHEMA, "strict": True},
            },
            messages=messages,
        )
        verdict = parse_judge_verdict(resp.choices[0].message.content)
        judge_fingerprint = getattr(resp, "system_fingerprint", None)

        rows = []
        now = datetime.now(timezone.utc).isoformat(timespec="seconds").replace("+00:00", "Z")
        for slot, cond in zip(("answer_a", "answer_b"), order):
            scores = verdict[slot]
            run_row = runs_by_cond[cond]
            det_cov = deterministic_citation_coverage(
                question.get("gold_cited_arxiv_ids", []),
                run_row.get("tool_calls", []),
            )
            rows.append({
                "scores_schema_version": SCORES_SCHEMA_VERSION,
                "question_id": question["question_id"],
                "condition": cond,
                "timestamp": now,
                "judge_model": JUDGE_MODEL,
                "judge_seed": JUDGE_SEED,
                "judge_temperature": JUDGE_TEMPERATURE,
                "judge_system_fingerprint": judge_fingerprint,
                "presentation_order": order,
                "presentation_slot": slot,
                "answer_correctness": scores["answer_correctness"],
                "faithfulness": scores["faithfulness"],
                "citation_coverage": scores["citation_coverage"],
                "completeness": scores["completeness"],
                "judge_notes": scores.get("notes", ""),
                "deterministic_citation_coverage": det_cov,  # D-19
                "error": None,
            })
        return rows


    # ---------- Main loop ----------

    def score_all(
        questions_path: Path = DEFAULT_QUESTIONS_PATH,
        runs_path: Path = DEFAULT_RUNS_PATH,
        scores_path: Path = DEFAULT_SCORES_PATH,
        rubric_path: Path = DEFAULT_RUBRIC_PATH,
        *,
        client_factory: Optional[Callable[[], object]] = None,
        limit: Optional[int] = None,
        dry_run: bool = False,
    ) -> dict:
        with open(questions_path) as f:
            doc = json.load(f)
        questions_by_id = {q["question_id"]: q for q in doc["questions"]}
        runs_by_pair = _load_runs_by_qid_cond(runs_path)
        done = _load_done_questions(scores_path)
        rubric_text = _read_rubric(rubric_path)

        # Only score questions that have BOTH success runs.
        pending: list[str] = []
        for qid in questions_by_id:
            if qid in done:
                continue
            if (qid, "baseline") in runs_by_pair and (qid, "citation_aware") in runs_by_pair:
                pending.append(qid)

        if limit is not None:
            pending = pending[:limit]

        logger.info("score_all: %d pending, %d already done", len(pending), len(done))
        if dry_run:
            return {"pending": len(pending), "done": len(done)}

        if client_factory is None:
            if not os.environ.get("OPENAI_API_KEY"):
                raise RuntimeError("OPENAI_API_KEY not set")
            from openai import OpenAI  # lazy
            client_factory = lambda: OpenAI()
        client = client_factory()

        written = 0
        for qid in pending:
            q = questions_by_id[qid]
            try:
                rows = score_question(

                    "question_id": qid,

                }
                _append_row(scores_path, err_row)
                logger.warning("score %s errored: %s", qid, err_row["error"])
        return {"written": written, "pending": len(pending), "done": len(done)}

    Inputs:
      - eval/results/scores.jsonl    (from eval/score.py)
      - eval/results/runs.jsonl      (from eval/run_eval.py — for tool_call / latency stats)
      - eval/questions.json          (for per-type breakdowns)

    Output:
      - eval/FINDINGS.md with 8 sections per D-20 + Wilcoxon (08-RESEARCH.md Pattern 6)
      + LLM-vs-deterministic citation_coverage agreement (D-19)
    """

    import argparse
    import json
    import logging
    import os
    import sys
    from collections import defaultdict
    from pathlib import Path
    from typing import Optional

    sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))

    DIMENSIONS = ("answer_correctness", "faithfulness", "citation_coverage", "completeness")
    DEFAULT_SCORES_PATH = Path(__file__).parent / "results" / "scores.jsonl"
    DEFAULT_RUNS_PATH = Path(__file__).parent / "results" / "runs.jsonl"
    DEFAULT_QUESTIONS_PATH = Path(__file__).parent / "questions.json"
    DEFAULT_OUTPUT_PATH = Path(__file__).parent / "FINDINGS.md"
    DEFAULT_FIGS_DIR = Path(__file__).parent / "notebook" / "figs"

    logger = logging.getLogger("analyze")
    logging.basicConfig(level=logging.INFO, format="%(asctime)s %(levelname)s %(message)s")


    def _load_jsonl(path: Path) -> list[dict]:
        rows = []
        with open(path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                row = json.loads(line)
                if row.get("error"):
                    continue
                rows.append(row)
        return rows


    def paired_deltas(scores: list[dict], dimension: str) -> list[tuple[str, float]]:
        """Returns [(question_id, citation_aware - baseline), ...] for every question with both conditions scored."""
        by_qid: dict[str, dict[str, float]] = defaultdict(dict)
        for r in scores:
            by_qid[r["question_id"]][r["condition"]] = r.get(dimension)
        out = []
        for qid, d in sorted(by_qid.items()):
            if "baseline" in d and "citation_aware" in d and d["baseline"] is not None and d["citation_aware"] is not None:
                out.append((qid, float(d["citation_aware"]) - float(d["baseline"])))
        return out


    def summarize_condition(scores: list[dict], condition: str) -> dict[str, dict]:
        """Per-dimension mean/median/stdev for a condition."""
        import statistics
        out = {}
        for dim in DIMENSIONS:
            vals = [float(r[dim]) for r in scores if r["condition"] == condition and r.get(dim) is not None]
            if vals:
                out[dim] = {
                    "n": len(vals),
                    "mean": round(statistics.mean(vals), 3),
                    "median": statistics.median(vals),
                    "stdev": round(statistics.stdev(vals), 3) if len(vals) > 1 else 0.0,
                }
            else:
                out[dim] = {"n": 0, "mean": None, "median": None, "stdev": None}
        return out


    def wilcoxon_test_four_dims(scores: list[dict]) -> dict[str, dict]:
        """Paired Wilcoxon signed-rank test on (citation_aware - baseline) for each dimension per 08-RESEARCH.md Pattern 6.

        Returns {dim: {"n": int, "median_delta": float, "stat_two_sided": float, "p_two_sided": float,
                       "stat_greater": float, "p_greater": float}}.
        Uses zero_method="wilcox" (drops zero-deltas) per scipy docs.
        When all deltas are zero / n<2, reports p=1.0.
        """
        import statistics
        from scipy.stats import wilcoxon
        out = {}
        for dim in DIMENSIONS:
            deltas = [d for _, d in paired_deltas(scores, dim)]
            nonzero = [d for d in deltas if d != 0.0]
            if len(nonzero) < 2:
                out[dim] = {
                    "n": len(deltas), "n_nonzero": len(nonzero),
                    "median_delta": statistics.median(deltas) if deltas else 0.0,
                    "stat_two_sided": None, "p_two_sided": 1.0,
                    "stat_greater": None, "p_greater": 1.0,
                    "note": "insufficient non-zero deltas for wilcoxon",
                }
                continue
            stat2, p2 = wilcoxon(deltas, zero_method="wilcox", alternative="two-sided")
            statg, pg = wilcoxon(deltas, zero_method="wilcox", alternative="greater")
            out[dim] = {
                "n": len(deltas), "n_nonzero": len(nonzero),
                "median_delta": round(statistics.median(deltas), 3),
                "stat_two_sided": float(stat2), "p_two_sided": float(p2),
                "stat_greater": float(statg), "p_greater": float(pg),
            }
        return out


    def deterministic_agreement(scores: list[dict]) -> dict:
        """Correlate judge's citation_coverage (1-5) with deterministic_citation_coverage (0-1) per D-19.

        Reports Spearman correlation, Pearson correlation (after rescaling judge to [0,1] by (x-1)/4),
        and a 'direction_agreement' metric: fraction of rows where judge high (>=4) iff det high (>=0.5).
        """
        try:
            from scipy.stats import spearmanr, pearsonr
        except ImportError:
            return {"error": "scipy unavailable"}
        judge = [float(r["citation_coverage"]) for r in scores if r.get("citation_coverage") is not None and r.get("deterministic_citation_coverage") is not None]
        det = [float(r["deterministic_citation_coverage"]) for r in scores if r.get("citation_coverage") is not None and r.get("deterministic_citation_coverage") is not None]
        if len(judge) < 3:
            return {"n": len(judge), "spearman_r": None, "spearman_p": None, "pearson_r": None, "pearson_p": None, "direction_agreement": None}
        sp_r, sp_p = spearmanr(judge, det)
        judge_rescaled = [(x - 1) / 4 for x in judge]
        pe_r, pe_p = pearsonr(judge_rescaled, det)
        agree = sum(1 for j, d in zip(judge, det) if (j >= 4) == (d >= 0.5)) / len(judge)
        return {
            "n": len(judge),
            "spearman_r": round(float(sp_r), 3), "spearman_p": float(sp_p),
            "pearson_r": round(float(pe_r), 3), "pearson_p": float(pe_p),
            "direction_agreement": round(agree, 3),
        }


    def by_question_type(scores: list[dict], questions: list[dict]) -> dict[str, dict]:
        """Per-question-type mean deltas for each dimension per D-03 stratification."""
        qtype_by_qid = {q["question_id"]: q["question_type"] for q in questions}
        out: dict[str, dict] = {}
        for qtype in ("method-dependency", "comparative", "claim-grounding"):
            qids_in_type = {qid for qid, t in qtype_by_qid.items() if t == qtype}
            sub_scores = [r for r in scores if r["question_id"] in qids_in_type]
            out[qtype] = {
                "n_questions": len(qids_in_type),
                "by_dim": {dim: [(qid, d) for qid, d in paired_deltas(sub_scores, dim)] for dim in DIMENSIONS},
            }
        return out


    def _latency_stats(runs: list[dict]) -> dict[str, dict]:
        import statistics
        out = {}
        for cond in ("baseline", "citation_aware"):
            lat = [float(r["latency_s"]) for r in runs if r.get("condition") == cond and r.get("latency_s") is not None and not r.get("error")]
            tok = [int(r["tokens_used"]) for r in runs if r.get("condition") == cond and r.get("tokens_used") is not None and not r.get("error")]
            out[cond] = {
                "latency_mean_s": round(statistics.mean(lat), 3) if lat else None,
                "latency_median_s": statistics.median(lat) if lat else None,
                "tokens_mean": round(statistics.mean(tok), 1) if tok else None,
                "n_rows": len(lat),
            }
        return out

        det_agree = deterministic_agreement(scores)
        by_qt = by_question_type(scores, questions)

        lines: list[str] = []

        w("")

        w(f"- Judge: `gpt-4o-mini`, temperature 0.0, seed 42 (D-11 / D-16).")
        w(f"- Questions scored: {n_scored} / {n_questions}.")
        for dim in DIMENSIONS:
            d = wilc[dim]
            dir_ = "citation_aware > baseline" if (d.get("median_delta") or 0) > 0 else ("citation_aware < baseline" if (d.get("median_delta") or 0) < 0 else "tied")
            w(f"- **{dim}**: median Δ = {d.get('median_delta')}, Wilcoxon two-sided p = {d.get('p_two_sided'):.4f}, one-sided (greater) p = {d.get('p_greater'):.4f} — {dir_}.")

        w("")

        for t, n in sorted(type_counts.items()):

        w("")
