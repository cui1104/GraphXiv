```python
import logging
import re
from sickle import Sickle
from sqlalchemy.dialects.postgresql import insert as pg_insert
from app.crawler.utils import (
    save_crawl_state, load_crawl_state, is_already_ingested, PMC_OAI_BASE,
)
from app.models import Paper, PaperSource
from app.db import SessionLocal

logger = logging.getLogger(__name__)

# Keywords to filter DL papers from the broad pmc-open set
DL_KEYWORDS = re.compile(
    r"deep learning|neural network|transformer|convolutional|recurrent neural|"
    r"attention mechanism|generative adversarial|reinforcement learning|"
    r"language model|BERT|GPT|diffusion model|graph neural",
    re.IGNORECASE,
)
```

1. **`def _extract_pmc_id(identifier: str) -> str`**:
   - Input: `"oai:pubmedcentral.nih.gov:PMC1234567"`
   - Return: `"PMC1234567"` (split on `:`, take last element)

2. **`def _is_dl_paper(title: str | None, abstract: str | None) -> bool`**:
   - Return True if DL_KEYWORDS matches in title or abstract
   - Return False if both are None

3. **`def harvest_pmc_ids(from_date: str = "2020-01-01", max_records: int = 50000) -> list[str]`**:
   - **Phase 1: Fast ID harvest** (avoids token expiry during slow processing)
   - Load crawl state for source `"pmc"` via `load_crawl_state`
   - If resumption_token exists, resume: `sickle.ListRecords(resumptionToken=token)`
   - Otherwise fresh: `sickle.ListRecords(metadataPrefix="pmc_fm", set="pmc-open", **{"from": from_date})`
   - Create `Sickle(PMC_OAI_BASE, timeout=60)` instance
   - Iterate records, collecting PMC IDs into a list
   - After every 10 records (one page boundary at PMC's 10/page), persist token:
     ```python
     token = getattr(records, 'resumption_token', None)
     if token:
         token_str = token.token if hasattr(token, 'token') else str(token)
         save_crawl_state(session, "pmc", token_str, record_count=10)
     ```
   - Stop at max_records or when token is exhausted
   - Return list of PMC IDs
   - **Critical:** Use `metadataPrefix="pmc_fm"` for harvest (front matter only, faster), NOT `"pmc"` (full JATS XML)
   - Log progress every 100 records: `"PMC harvest: {count} IDs collected"`

4. **`def process_pmc_record(session, pmc_id: str, title: str | None = None, abstract: str | None = None) -> bool`**:
   - Skip if `is_already_ingested(session, pmc_id=pmc_id)` returns True
   - If title/abstract provided, check `_is_dl_paper(title, abstract)` — skip if False
   - Create Paper row: `pmc_id=pmc_id, title=title, abstract=abstract`
   - Use `pg_insert(Paper).values(...).on_conflict_do_nothing(index_elements=["pmc_id"])` for safety
   - Create PaperSource row: `source_type="pmc", parse_status="pending"`
   - Return True if inserted, False if skipped

5. **`def harvest_pmc(from_date: str = "2020-01-01", max_records: int = 50000) -> int`**:
   - Main entry point. Two-phase approach:
   - **Phase 1:** Call `harvest_pmc_ids(from_date, max_records)` to get all IDs
   - **Phase 2:** For each ID, fetch metadata (title, abstract) using a second sickle call with `metadataPrefix="pmc_fm"` and `identifier=pmc_id`, OR extract from the initial harvest if available in record.metadata
   - Call `process_pmc_record(session, pmc_id, title, abstract)` for each
   - Commit every 50 records
   - Log final count
   - Return total inserted count
   - **Simplification:** If `pmc_fm` metadata already includes title/abstract in the harvest records, extract them inline during Phase 1 instead of making a second pass. Check `record.metadata` dict for keys like `article-title` or `dc:title`.
