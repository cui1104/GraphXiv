2. **`_fetch_page(client, params) -> httpx.Response`** — async, rate-limited, retried:
   - Decorate with `@retry(stop=stop_after_attempt(5), wait=wait_exponential(multiplier=2, min=4, max=60), retry=retry_if_exception_type((httpx.HTTPStatusError, httpx.TimeoutException)), reraise=True)`
   - Inside: `async with RATE_LIMITER:` then `resp = await client.get(ARXIV_OAI_BASE, params=params, headers={"User-Agent": USER_AGENT}, timeout=30.0, follow_redirects=True)`
   - Call `resp.raise_for_status()`, return resp

3. **`_parse_arxiv_records(xml_text: str) -> list[dict]`**:
   - Parse with `etree.fromstring(xml_text.encode())`
   - Find all `oai:ListRecords/oai:record` elements using namespace-aware xpath
   - For each record, extract from the metadata element (namespace varies — look for child elements):
     - `id` → normalize with `normalize_arxiv_id()`
     - `title` → strip whitespace
     - `abstract` → strip whitespace
     - `authors` → as string
     - `categories` → space-separated string
     - `doi` → if present
     - `created` → parse year from date string (first 4 chars)
   - Return list of dicts with keys: `arxiv_id`, `title`, `abstract`, `authors`, `categories`, `doi`, `year`
   - **Critical:** arXivRaw metadata is NOT in the OAI namespace — it uses no namespace or the arXiv namespace. Use `record.find('.//id')` or iterate child elements if namespace-aware xpath fails. Log and skip records that fail to parse.

4. **`_extract_resumption_token(xml_text: str) -> str | None`**:
   - Find `oai:ListRecords/oai:resumptionToken` element
   - Return `.text.strip()` if non-empty, else None
   - Note: post-March 2025 arXiv tokens have NO `completeListSize` or `cursor` attributes

5. **`async def harvest_arxiv_set(set_name: str, from_date: str = "2020-01-01") -> int`**:
   - Create `SessionLocal()` session
   - Load existing crawl state via `load_crawl_state(session, f"arxiv:{set_name}")`
   - If token exists in crawl state, resume with `params = {"verb": "ListRecords", "resumptionToken": token}`
   - Otherwise start fresh: `params = {"verb": "ListRecords", "set": set_name, "metadataPrefix": "arXivRaw", "from": from_date}`
   - Loop: fetch page, parse records, for each record:
     - Skip if `is_already_ingested(session, arxiv_id=record["arxiv_id"])` returns True
     - Otherwise: create Paper row (arxiv_id, title, abstract, year, doi from record), flush to get canonical_id
     - Create PaperSource row (canonical_id=paper.canonical_id, source_type="arxiv", parse_status="pending")
     - **Enqueue asset download** (INGEST-02): After inserting each new paper, call:
       ```python
       from app.tasks.ingest import download_asset
       download_asset.apply_async(args=[record["arxiv_id"], "arxiv"], queue="fast")
       ```
       Import `download_asset` at the top of the function (lazy import to avoid circular import at module level). This ensures every newly harvested paper gets its e-print asset downloaded. Per RESEARCH.md "Celery Task Enqueue from Crawler" pattern.
     - Commit batch
   - After each page: `save_crawl_state(session, f"arxiv:{set_name}", token, record_count=len(new_records))`
   - Extract resumption token; if None, harvest complete — save crawl_state with token=None
   - Log progress: `"Set {set_name}: page done, {count} new records, token={'present' if token else 'none'}"`
   - Return total new records inserted
   - **Handle duplicate arxiv_id:** Use `INSERT ... ON CONFLICT (arxiv_id) DO UPDATE SET title=..., abstract=..., updated_at=now()` via SQLAlchemy `pg_insert` to handle v2 re-ingestion (INGEST-05).

6. **`async def harvest_all_arxiv(from_date: str = "2020-01-01") -> dict[str, int]`**:

   - Return dict mapping set_name -> record count
   - Use `httpx.AsyncClient()` context manager shared across all sets (sequential, not parallel — respect rate limit)

2. Update `app/tasks/ingest.py` — replace stubs with real implementations that handle BOTH arXiv and PMC sources:

   ```python
   import asyncio
   from celery import shared_task
   from app.db import SessionLocal
   from app.models import Paper, PaperSource
   from app.crawler.arxiv_oai import harvest_arxiv_set
   from app.crawler.arxiv_assets import download_eprint_asset

   @shared_task(bind=True, name="app.tasks.ingest.ingest_paper", max_retries=3, time_limit=300, soft_time_limit=280, default_retry_delay=30)
   def ingest_paper(self, paper_id: str, source: str = "arxiv") -> dict:
       """Harvest metadata for a single set or trigger a full harvest."""
       try:
           if source == "arxiv":
               count = asyncio.run(harvest_arxiv_set(paper_id))  # paper_id is set_name for bulk
               return {"status": "success", "source": source, "set": paper_id, "records": count}
           elif source == "pmc":
               # Lazy import to avoid circular imports and allow 02-03 to implement pmc_oai.py independently
               from app.crawler.pmc_oai import harvest_pmc
               count = harvest_pmc(from_date=paper_id)  # paper_id is from_date for PMC bulk
               return {"status": "success", "source": source, "records": count}
           else:
               return {"status": "unknown_source", "source": source}
       except Exception as exc:
           self.retry(exc=exc)

   @shared_task(bind=True, name="app.tasks.ingest.download_asset", max_retries=3, time_limit=120, soft_time_limit=110, default_retry_delay=15)
   def download_asset(self, paper_id: str, source_type: str) -> dict:
       """Download e-print asset for a given arXiv ID and update PaperSource."""
       try:
           asset_path, detected_type = asyncio.run(download_eprint_asset(paper_id))
           session = SessionLocal()
           try:
               paper = session.query(Paper).filter(Paper.arxiv_id == paper_id).first()
               if paper:
                   ps = session.query(PaperSource).filter(
                       PaperSource.canonical_id == paper.canonical_id,
                       PaperSource.source_type == "arxiv"
                   ).first()
                   if ps:
                       ps.asset_path = asset_path
                       ps.source_type = detected_type
                       session.commit()
           finally:
               session.close()
           return {"status": "success", "paper_id": paper_id, "asset_path": asset_path, "source_type": detected_type}
       except Exception as exc:
           self.retry(exc=exc)
   ```

   Note: `time_limit` increased to 300s for ingest_paper (harvests can take minutes) and 120s for download_asset (large files). The PMC branch uses a lazy import of `harvest_pmc` from `app.crawler.pmc_oai` — this allows 02-03 to create pmc_oai.py independently while 02-02 wires the Celery task graph. If pmc_oai.py does not exist yet at import time, the lazy import inside the function body ensures no ImportError at module load.
